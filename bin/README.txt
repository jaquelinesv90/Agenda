-O exercício é fazer um mini projeto de agendamentos ( Java SE ).

-Requisitos:

-O sistema deve apresentar um menu de opções ao usuário. 
Se o cliente digitar o nro 1, o sistema deve solicitar os seguintes dados ao usuário:

	-Nome do Cliente;
	-Telefone;
	-CPF;
	-Rua;
	-Nro;
	-Bairro;
	-Cidade;
	-Descrição do Serviço agendado;
	-Horário desejado;

-Se o cliente digitar o nro 2, o sistema deve solicitar o nome do cliente e
 em seguida apresentar os agendamentos somente desse cliente cujo nome foi digitado anteriormente.
 Os dados a serem apresentados em tela são: 

    -Nome do Cliente;
    -Telefone;
    -Descrição do Serviço agendado;
    -Horário desejado;


-Regras de Negócio:

- Os dados dos clientes devem ser armazenados em um arraylist onde cada "nó" representa um objeto do tipo cliente.
